# d-quant-visualizer

Python tools to visualize MIDI performance data generated by the [d-quant](https://github.com/yourusername/d-quant) system.

## Setup

```bash
pip install -r requirements.txt
```

## Usage
```bash
python scripts/plot_envelope.py
```
